What is Solid Framework

Solid Framework is a small boilerplate designed to quickly start new projects. Yet offering more features than most boilerplates. These are features I personally use in all my project. So much that I just decided to add them in, saving me atleast an hour or two on each project. One of Solid Framework its main features is a non floating grid system. It's a grid system that stays within the flow of the website, making sure that your website or app has a solid base.

Equal height columns without using Flexbox

Sometimes you're in need of some columns, columns that are equally in height. With Flexbox it's super easy to get your heights equal across all columns in a row. However, sometimes you're dealing with a client who is forced to use an older browser, Internet explorer 9 for example. For these clients, there is Solid. Solid fakes the height of an element, forcing them to look like they're equally in height.

To make optimal use of the equal height columns, make sure you'll give the borders the correct colour, otherwise you're going to have a bad time.
